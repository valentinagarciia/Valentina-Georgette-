Los pasos para crear un repositorio y publicar en Git hub son:
1. Ir a https://github.com
2. Haz clic en Sign up
3. Completa el registro
4. Haz clic en el botón "New" o en el símbolo + (arriba a la derecha): New repository
5. Completa: Repository name (nombre del proyecto)
6. Haz clic en Create repository
7. Opción A: Si ya tienes un proyecto creado
- Abre la terminal dentro de la carpeta del proyecto y ejecutar
- Luego conecta tu proyecto con GitHub
8. Opción B: Clonar el repositorio vacío desde GitHub
- Luego agrega archivos
9. Actualiza la página del repositorio en GitHub y deberías ver tus archivos publicados
